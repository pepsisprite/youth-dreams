### 微电影简介       
学生林有，自小就有一个导演梦，但他内心孤僻，与同学关系冷淡。    
梦想与学习冲突不断，母亲的怒斥，老师的训诫，无一不刺激着他的神经。     
直到蒋无等人的出现……    
从无到有，从孤僻到开朗，从冷漠到活泼，他们究竟碰撞出了怎样的火花？不堪入目的成绩又是如何出现大逆转？在奔向未来的途中，一场梦想与现实的风暴正在席卷而来。
